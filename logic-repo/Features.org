#+TITLE: Features

This document describes the various feature TLVs which can be embedded in a
/Config/ file type. Refer to the /Configuration File/ specification for a
definition of the containing file structure for the feature TLVs.

* Change Serial Number
Change the serial number of the device.

| Field  | Length | Value                     |
|--------+--------+---------------------------|
| TAG    |      1 | ~0xE2~                    |
| LENGTH |      1 | ~0x08~                    |
| S/N    |      8 | New reader serial number. |

* Prox Filter
Configure the prox/bitstream filter.

| Field          | Length | Value                     |
|----------------+--------+---------------------------|
| TAG            |      1 | ~0xFC~                    |
| LENGTH         |      1 | ~0x12~                    |
| Applied Filter |      1 | Applied filter bitmask.   |
| Num Bits       |      1 | Number of bits in filter. |
| Filter Mask    |      8 | Prox filter mask.         |
| Filter Value   |      8 | Prox filter value.        |

** Applied Filter
The applied filter field can have the following values applied as a bitmask.

/Note: The lower nibble chooses the filter technology. The upper nibble chooses function./

| Field        | Value  |
|--------------+--------|
| Filter HF    | ~0x01~ |
| Filter LF    | ~0x02~ |
| Filter BLE   | ~0x04~ |
| Filter CSN   | ~0x08~ |
| Equal to     | ~0x00~ |
| Greater than | ~0x40~ |
| Less than    | ~0x80~ |
| Not Equal to | ~0xC0~ |

* Reset to Loaded Config
Resets all settings to their originally loaded values from the factory.

| Field  | Length | Value  |
|--------+--------+--------|
| TAG    |      1 | ~0xFB~ |
| LENGTH |      1 | ~0x00~ |

* Reset to Standard Features
Reset to absolute default settings (Pre-factory config).

*This is only available if reader is in developer mode.*

| Field  | Length | Value  |
|--------+--------+--------|
| TAG    |      1 | ~0xF0~ |
| LENGTH |      1 | ~0x00~ |

* Reset to Standard Keys
Reset to absolute standard keys.

*This is only available if reader is in developer mode.*

| Field  | Length | Value  |
|--------+--------+--------|
| TAG    |      1 | ~0xF8~ |
| LENGTH |      1 | ~0x00~ |

* Full Feature Setup
Performs a "full configuration" overwriting the entire ~SystemFeatures~ structure in the
reader.

*NOTE:* If the reader is in developer mode and this TLV is passed with a 0 length. The
full configuration will be written to the next config card presented to the reader.

| Field           | Length | Value                                       |
|-----------------+--------+---------------------------------------------|
| TAG             |      1 | ~0xF1~                                      |
| LENGTH          |      3 | ~0x820400~                                  |
| System Features |   1024 | The entire ~SystemFeatures~ data structure. |

* ECP NFC Setup
Configures the parameters in the Apple ECP 14443-A polling structure.

*NOTE:* These values will be provided by Apple in most cases, but a few of the fields
remain somewhat fixed initially.

| Field                 | Length | Value                           |
|-----------------------+--------+---------------------------------|
| TAG                   |      1 | ~0xD2~                          |
| LENGTH                |      1 | ~0x07~                          |
| ECP format            |      1 | ~0x02~                          |
| ECP terminal info     |      1 | ~0xC3~ or ~0x83~ for user auth. |
| ECP terminal type     |      1 | ~0x02~                          |
| ECP terminal sub-type |      1 | See Subtypes.                   |
| TCI 1                 |      1 | ~0x02~                          |
| TCI 2                 |      1 | End user specific               |
| TCI 3                 |      1 | End user specific               |

** Sub Types

| University  | ~0x00~ |
| Corporate   | ~0x01~ |
| Hospitality | ~0x02~ |
| Residentail | ~0x03~ |

* Android NFC Setup
Configure the Android NFC settings for LFM credentials.

| Field        | Length | Value                                    |
|--------------+--------+------------------------------------------|
| TAG          |      1 | ~0xD2~                                   |
| LENGTH       |      1 | ~0x46~                                   |
| NFC Settings |     70 | The entire ~nfcSettings~ data structure. |

* Operational Settings
Modify the operational settings. This TLV performs individual modifications
to fields within the ~OpSettings~ structure in the reader.

| Field         | Length | Value                              |
|---------------+--------+------------------------------------|
| TAG           |      1 | ~0xF3~                             |
| LENGTH        |      1 | ~0x04~                             |
| Bit operation |      1 | See bit operations.                |
| Option index  |      1 | Index to perform operation on.     |
| Option data   |      2 | Mask to use with the bit operation |

** Bit Options

| Option     | Value  |
|------------+--------|
| Bit Clear  | ~0x01~ |
| Bit Set    | ~0x02~ |
| Bit Toggle | ~0x03~ |

* RF Protocols
Modify the supported RF protocols. This will overwrite the current card protocols
configured in the reader.

| Field        | Length | Value                        |
|--------------+--------+------------------------------|
| TAG          |      1 | ~0xF4~                       |
| LENGTH       |      1 | ~0x01~ to ~0x10~             |
| RF Protocols |   1-16 | New RF protocols to support. |

* Card Apps
Modify the supported smart card/BLE applications. This will overwrite the current
supported card types.

| Field      | Length | Value                      |
|------------+--------+----------------------------|
| TAG        |      1 | ~0xF5~                     |
| LENGTH     |      1 | ~0x01~ to ~0x08~           |
| Card Types |    1-8 | New card types to support. |

* Keyset
Load a keyset into the reader.

| Field         | Length | Value                                         |
|---------------+--------+-----------------------------------------------|
| TAG           |      1 | ~0xF6~                                        |
| LENGTH        |      1 | ~0x20~                                        |
| Keyset Number |      1 | The keyset number to overwrite in the reader. |
| Keyset        |     32 | Two AES128 keys to overwrite in keyset slot.  |

* Force Software Reset
Forces the reader to perform a full reset.

| Field  | Length | Value  |
|--------+--------+--------|
| TAG    |      1 | ~0xEB~ |
| LENGTH |      1 | ~0x00~ |

* Rotate Option
Rotate/toggle a configuration option.

| Field  | Length | Value  |
|--------+--------+--------|
| TAG    |      1 | ~0xFD~ |
| LENGTH |      1 | ~0x01~ |
| Option |      1 | See options. |

** Options

| Rotate CASI 4001/4002       | ~0x01~ |
| Rotate F2F supervision mode | ~0x02~ |
